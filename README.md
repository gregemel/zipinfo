# Zip Info Service
* provide US zip code
* get back temperature, time zone, and elevation
* uses:
    * Open WeatherMap
    * Google Time Zone
    * Google Elevation
    
#### for Cayuse code challenge 

sample usage:
* run service on port 8080 (default)
* using browser:
    * http://localhost:8080/api/v1/zipinfo/97201
